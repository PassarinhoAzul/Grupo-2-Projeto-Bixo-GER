## Procedimento 

A primeira coisa que fizemos foi medir os componentes que vamos utilizar no carrinho, sendo elas:
- ESP32-C3 Mini: 18mm x 22mm 4mm
- Sensor Ultrassônico HC-SR04: 45mm x 20mm x 15mm
- Ponte H L298N: 43mm x 43mm x 27
- Motor TT: 22mm x 70mm x 18mm
- Roda 65mm: 65mm x 27mm x 65mm
- Suporte para as pilhas: 75mm x 40mm x 20mm
- Roda Boba: 30mm x 30mm x ---

Após isso, fizemos uma reunião para debater as ideias iniciais para o modelo, buscando inspirações em carros de formula um e outros projetos semelhantes. O rascunho desenvolvido está representado na figura abaixo. 

![Rascunho](/imagens/Rascunho.png)



Durante a modelagem do projeto, baixamos modelos 3d dos componentes, obtidos no site ([3d Content Central](https://www.3dcontentcentral.com/Default.aspx)) para fazer testes com o modelo e verificar diferentes possibilidades para o posicionamento dos objetos no chassi.

### Chassi:

As medidas do Chassi são de 100mm x 160m x 3m. Utilizamos essaas medidas baseada no encaixe dos componentes, com uma espessura adequada para suportar o peso de todos. Detalhando mais sobre o projeto, escolheu-se colocar os dois motores na parte de baixo e o restante dos componentes na parte de cima do chassi, reservando um espaço para um possível sensor ultrassônico. Além disso, o projeto possui um furo no centro para facilitar as conexões entre os componentes superiores e inferiores do carro. Por fim, os furos necessários para conectar os componentes ao chassi serão feitos utilizando uma furadeira, devido a facilidade e liberdade em compaação a criar furos diretamente no modelo das peças. ### Aerofólio

O aerofólio é composto por duas partes: dois suportes laterais (30mm x 40mm x 3mm) e uma peça de encaixe horizontal (60mm x 10mm x 3mm). A tolerância para o encaixe de cada peça é de 0.5mm. Essa parte será encaixada e fortificada com cola no chassi. Abaixo está uma imagem contendo o aerofólio montado e cada peça individual.
